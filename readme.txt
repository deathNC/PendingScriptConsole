PendingScriptConsole
====================

Neiro Technology 2011-2014

Java-sources of MultiTrading platform Pending Script Console

Данная платформа предназначена для тогровли на нескольких терминалах MetaTrader4 диллерского центра Forex-Inn. Передача данных основана на TCP/IP протоколе. Но юзать её можно всё равно только локально, потому что некоторые, жизненно необходимые компоненты не будут работать удалённо.
